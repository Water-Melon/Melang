### Prime Number



##### mln_prime

Generate a prime number greater than or equal to a given number.

```
mln_prime(base);
```

Input:

- base - an integer number.

Return value:

- a prime number.

Error:

- Invalid argument will throw an error.

Example:

```
mln_print(mln_prime(22));
```

The output is:

```
23
```
